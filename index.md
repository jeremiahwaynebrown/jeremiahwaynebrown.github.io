## Professional Self-Assessment

Throughout my time in the Computer Science program at Southern New Hampshire University, I have developed a strong foundation in software development, problem solving, and system design. Completing this capstone and building my ePortfolio has allowed me to reflect on that growth while also demonstrating my ability to apply these skills in real-world scenarios. Each artifact included in this portfolio represents a different area of computer science and highlights both my technical abilities and my progression as a developer.

One of the most important skills I have developed is the ability to design and improve software systems. Through my OpenGL scene project, I demonstrated my understanding of software engineering principles by refactoring a previously static and rigid codebase into a more modular and scalable system. By introducing abstractions such as reusable objects and improving the structure of the rendering process, I was able to create a more maintainable and extensible design. This experience reinforced the importance of clean architecture and thoughtful code organization, which are essential in professional development environments.

In addition to software design, I strengthened my understanding of algorithms and data structures through the course planner artifact. In this project, I implemented more efficient data handling using structures such as hash maps and incorporated recursive logic to analyze prerequisite relationships. These enhancements allowed the program to handle more complex scenarios while improving performance and usability. This experience helped me better understand how algorithmic decisions directly impact the efficiency and scalability of an application.

My work with databases and full-stack concepts is demonstrated through the MongoDB dashboard project. In this artifact, I built a system that connects a database to a user-facing dashboard using Python and Dash. I implemented full CRUD functionality and created dynamic filtering and visualization features that allow users to interact with real data in meaningful ways. This project strengthened my ability to integrate multiple technologies and highlighted the importance of designing systems that are both functional and user-friendly.

Throughout the program, I have also gained experience communicating technical concepts and working with stakeholders. Whether through written documentation, discussion posts, or project explanations, I have learned how to present complex ideas in a clear and understandable way. This is especially important in collaborative environments, where developers must communicate effectively with both technical and non-technical audiences.

Security and data integrity have also become an important part of my approach to development. In working with databases and user-driven applications, I have learned to consider validation, error handling, and responsible data management as essential components of any system. These considerations ensure that applications are not only functional but also reliable and safe for users.

This ePortfolio brings together my work across software engineering, algorithms, and database development into a single, cohesive presentation. Each artifact builds on the others and demonstrates a different aspect of my skill set, while collectively showing my readiness to enter the computer science field. Through this process, I have gained confidence in my ability to analyze existing systems, implement meaningful improvements, and deliver complete, functional solutions.

As I move forward in my career, I plan to continue building on these skills and expanding my knowledge in full-stack development and software design. This portfolio serves not only as a reflection of my academic journey but also as a foundation for my future growth as a developer.
